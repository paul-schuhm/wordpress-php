## Bibliographie pour aller plus loin

### WordPress

- [Professional WordPress: Design and Development, 3rd Edition](https://www.amazon.com/Professional-WordPress-Development-Brad-Williams/dp/1118987241) de Brad Williams et David Damstra, Edition Wrox, 3rd Edition, 2015
- [Professional WordPress Plugin Development, 2nd edition](https://www.amazon.com/Professional-WordPress-Plugin-Development-2nd/dp/1119666945) de Brad Williams et Justin Taldock, Edition Wrox, 2nd Edition, 2020

### PHP

- [Modern PHP: new features and good practices](https://www.oreilly.com/library/view/modern-php/9781491905173/), Josh Lochart, Edition O'REILLY, 2015. Excellent livre présentant l'approche du PHP *moderne* (> PHP7) **LP++**
- [Essential PHP Security](https://www.oreilly.com/library/view/essential-php-security/059600656X/), Chris Shiflett, O'Reilly Media, Inc., 2005. Excellente introduction à la sécurité des applications web et des applications web PHP en particulier, **concis** (moins de 100 pages !). **LP+**
